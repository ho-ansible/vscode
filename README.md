# Ansible role: vscode
Open-source build of MS VSCode IDE (vscodium).

## Requirements
Only tested on Debian stable, for now.

## Role Variables

## Playbooks
+ `main.yml`: apply role
+ `uninstall.yml`: remove. Run prior to removing host from inventory group.

## Dependencies
None.
